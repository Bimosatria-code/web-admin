# Web Admin



## Getting started
Rename ```.env.example``` to ``` .env ``` and setup db connection like below

```
    DB_CONNECTION=mssql
    DB_HOST=[Your SQL Server Host]
    DB_DATABASE=webadmin
    DB_USERNAME=[Your Username]
    DB_PASSWORD=[Your Pass]
```

```
    cd web-admin
    npm install
    npm start
    access to 127.0.0.1:3000
```
